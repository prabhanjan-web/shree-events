Netlify Drop not working? No problem — use any of these 3 free alternatives below. All give you a permanent public URL that works forever.

Deploy with Vercel (Recommended)
1
Go to vercel.com and sign up free with Google or GitHub
2
Click "Add New → Project" on the dashboard
3
Scroll down and click "Browse" under "Import Project" → select your php project folder from Desktop
4
Leave all settings as default and click "Deploy"
5
In ~30 seconds you'll get a URL like:
shree-events.vercel.app
6
Copy that URL → go to Generate QR tab → paste & generate your Lifetime QR
Deploy with Surge.sh (Fastest — No signup needed!)
1
Open Command Prompt (Win+R → type cmd → Enter)
2
Install surge (one time only):
npm install --global surge
3
Navigate to your project folder:
cd "C:\Users\Prabhanjan S\OneDrive\Desktop\php project"
4
Run surge:
surge . shree-events.surge.sh
5
Enter any email & password when asked (creates account automatically). Your site is live at:
shree-events.surge.sh — Free forever!
6
Copy that URL → Generate QR tab → paste & generate your Lifetime QR
Deploy with GitHub Pages (100% Permanent)
1
Create a free account at github.com
2
Click "New Repository" → name it shree-events → set to Public → click Create
3
Click "uploading an existing file" → drag your entire php project folder contents into the upload box
4
Click "Commit changes" (green button at bottom)
5
Go to Settings → Pages → Source → Deploy from branch → main → / (root) → Save
6
Wait 2 mins. Your site is live at:
yourusername.github.io/shree-events
7
Copy that URL → Generate QR tab → paste & generate your Lifetime QR
